Experience




As a software developer intern with the Digital Innovation Lab at Singapore Airlines, I was responsible for developing scalable applications by analysing clients' requirements. Following an Agile methodology along with a design-centric approach, I learnt how to adapt to ever evolving requirements and user stories.

I developed a hybrid web-view application using Node.js, ejs and Swift (for iOS) to enable digitalisation of recording business card details upon scanning, and saving them directly into respective person's contacts. In order to extract details, I performed Optical Character Recognition using Google Vision API in Node.js, which was then categorised into appropriate fields via Natural Language Processing using 'nltk' library along with my own algorithm in Python.

I successfully deployed the iOS application for beta testing on TestFlight, which accelerated recording business card details by 15 seconds per card.
