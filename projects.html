Projects





Developed an Android application Q, with an aim to digitalise queueing for stalls in food courts across Singapore, thereby enabling people to utilise their time productively which is otherwise wasted waiting in long queues just to order food. Using this application, stall merchants can create and manage queues for their stalls, while customers can view and join these queues at their own convenience. The app. notifies the customer when his/her turn is approaching.

Used Java, Object-Oriented Programming, Android Development concepts extensively on Android Studio; used Google Firebase as the app's cloud database for the back-end.





Created a multiplayer arcade game Bit Combat in which two players compete against each other by moving around a 7x7 LED matrix and shooting bullets at each other using buttons, while avoiding obstacles at the same time. First player to kill his/her opponent wins the game!

Used concepts from the beta architecture of a von Neumann machine to develop this game on Field Programmable Gate Array (FPGA) Board (Mojo V3) that utilises a 16-bit ALU developed using Lucid programming language on Mojo IDE

Bit Combat was chosen as the course's best project by the faculty and students, based on voting administered by the Office of Student Administration at SUTD, and is showcased to the general public during outreach events at SUTD e.g. SUTD Open House, Design Exhibition

Developed a Python application Leaf-Me-Alone, with an objective to monitor the growth of a plant in real-time and show the live image, moisture levels, and light intensity of the plant to its owner.

Used Kivy, a Python framework for NUI development, to create the app interface; Google Firebase to store and retrieve moisture levels, light intensity and live image at a recurring frequency of 5 seconds; used Raspberry-Pi for multiple sensors and camera used.
